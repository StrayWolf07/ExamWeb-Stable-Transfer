# SQLite Database Location

**Active database file:** `prisma/dev.db`  
(Relative to project root: `./prisma/dev.db`)

**Connection string in .env:** `DATABASE_URL="file:./dev.db"`  
(Path is relative to this prisma/ directory)

For transfer: Copy `prisma/dev.db` along with the project. Do not run migrations, db push, or migrate reset on an existing database.
